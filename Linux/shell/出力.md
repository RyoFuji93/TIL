# >
標準出力をディスプレイではなく、ファイルに書き出す（ファイルの中身が既に存在する場合は上書き）
# >>
標準出力をディスプレイではなく、ファイルに書き出す（ファイルの中身が既に存在する場合は追記）
# 2>
標準エラー出力をディスプレイではなく、ファイルに書き出す（ファイルの中身が既に存在する場合は上書き）
# 2>>
標準エラー出力をディスプレイではなく、ファイルに書き出す（ファイルの中身が既に存在する場合は追記）
# &>
標準出力と標準エラー出力をディスプレイではなく、ファイルに書き出す（ファイルの中身が既に存在する場合は上書き）
# &>>
標準出力と標準エラー出力をディスプレイではなく、ファイルに書き出す（ファイルの中身が既に存在する場合は上書き）
# 2>&1
標準エラー出力も標準出力と一緒の設定にする
# > /dev/null
標準出力をディスプレイにもファイルにも出力しない
# 2> /dev/null
標準エラー出力をディスプレイにもファイルにも出力しない
# &> /dev/null
標準出力も標準エラー出力もディスプレイにもファイルにも出力しない
