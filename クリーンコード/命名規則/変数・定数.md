# いい変数名とは
1. 変数が保持するものを正確に説明する。
2. 曖昧でない
3. 短すぎず、長すぎない変数

### BAD

```js
class Calculation {
  divide(minutes) {
    return minutes / 60;
  }
  multiply(minutes) {
    return minutes * 60;
  }
}
```
### Good

```js
class TimeCalculation {
  getHours(minutes) {
    return minutes / 60;
  }
  getSeconds(minutes) {
    return minutes * 60;
  }
}
```
