# 関数の命名規則
* 分かりやすい関数名にする。
### Bad

```js
// 単位を分から時間に変換する関数
function fn(num) {
  return num / 60;
}
```

### Good

```js
function convertMinsToHours(minutes) {
  return minutes / 60;
}
```

* 関数の目的を明確にするために動詞をつける。

値の取得にはget、設定にはsetをつけるなど。

* 真偽値を返す関数（true,false）を返す関数

⇒is、hasを、can、shouldを接頭語（prefix）で始める。

⇒存在有無は末尾にexistsをつける。

* 動詞（+形容詞+名詞）にすると良い
~~~ js
getDayOfWeek
getNumberArray
addActiveClass
~~~
